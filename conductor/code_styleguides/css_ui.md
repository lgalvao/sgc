# Guia de Estilo: CSS & UI (SGC)

## Princípios Gerais
- **Bootstrap 5:** Utilize preferencialmente classes utilitárias do Bootstrap 5 via BootstrapVueNext.
- **Responsividade:** Projete interfaces mobile-first ou que se adaptem perfeitamente a resoluções de notebooks corporativos.

## Cores e Identidade
- **Padronização:** Utilize as cores institucionais do tribunal.
- **Feedback Visual:** Utilize as cores semânticas do Bootstrap (`success`, `danger`, `warning`, `info`) para indicar as 'Situações' dos processos.

## Acessibilidade (WCAG 2.2)
- **Contraste:** Garanta contraste suficiente em todos os elementos de texto.
- **Foco:** Não remova os estilos de `:focus` sem fornecer uma alternativa visível.
- **Semântica:** Use tags HTML semânticas (main, nav, section, article) corretamente.
