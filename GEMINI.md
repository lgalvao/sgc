# Instruções para o Agente Gemini

Bem-vindo ao projeto SGC. Para começar, por favor siga estas instruções:

1. **Leia o `AGENTS.md`**: Este arquivo contém diretrizes essenciais e boas práticas para o desenvolvimento neste projeto.
2. **Explore a Documentação**:
    * Existe um `README.md` na raiz do repositório que fornece uma visão geral do sistema.
    * Existem vários arquivos `README.md` específicos de contexto (por exemplo, dentro de pacotes backend e diretórios frontend) que detalham a arquitetura e responsabilidades de cada parte do sistema.
    * Utilize esses documentos para obter um entendimento aprofundado da base de código.
